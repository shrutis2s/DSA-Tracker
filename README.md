# DSA Tracker For Beginners

A full-stack web application to track progress on the Data Structures and Algorithms Sheet containing 120 questions. Built to help developers maintain consistency in solving Data Structures and Algorithms problems.

![Project Status](https://img.shields.io/badge/Status-In%20Development-green)
![License](https://img.shields.io/badge/License-MIT-blue)

## ‚ú® Features
- **Curated Question List:** Includes the complete Striver 75 sheet (Arrays, Linked Lists, Trees, Graphs, etc.).
- **Progress Tracking:** Dynamic progress bar showing completion percentage.
- **Topic Filtering:** Filter questions by category (e.g., "Arrays", "Binary Search").
- **Note Taking:** Add personal notes (time complexity, logic) for each question.
- **Persistence:** Uses a MySQL database to save your progress and notes permanently.

## üöÄ How to Run Locally

### 1. Clone the Repo
```bash
git clone [https://github.com/YOUR_USERNAME/dsa-tracker.git](https://github.com/YOUR_USERNAME/dsa-tracker.git)
cd dsa-tracker
```

### 2. Setup Database (MySQL)
```bash
  -Open MySQL Workbench.
  -Create a database named dsa_tracker
  -Import the setup.sql file provided in the server folder.
```
### 3. Setup Backend
```bash
  -cd server
  -npm install
   # Configure your db.js with your MySQL password
  -npm run dev  
```
### 4. Setup Frontend
```bash
  -cd client 
  -npm install
  -npm start
```
